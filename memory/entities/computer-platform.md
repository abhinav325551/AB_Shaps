# Computer – Platform architecture

Source: devrev.ai/how-computer-works, /meet-computer, /agent-studio. Verified 2026-03-31.

## Computer (AI Teammate)
- Conversational interface – searches, answers, reasons, co-creates, acts autonomously
- GA: Feb 9, 2026 (beta Sept 2025). Desktop: ⌘+. / Ctrl+.
- Permission-aware CRUD: updates CRM, logs activities, creates tasks, routes approvals
- Dual retrieval: Text-to-SQL (structured) + vector embeddings (semantic)

## AirSync (Patented)
- Bidirectional sync across: Salesforce, Zendesk, Jira, Slack, Google Workspace, Notion, HubSpot, Gmail, Gong, ServiceNow, Okta, Linear, GitHub, Confluence, MS Teams/OneDrive/SharePoint
- Unifies structured (CRM, tickets) + unstructured (emails, Slack, call summaries)
- Near real-time on Max tier

## Memory (Patented)
- AI-native knowledge graph structured around products, customers, relationships
- Sub-200ms at petabyte scale, multi-region, distributed caching
- Permission-aware (RBAC + ABAC)
- Combines multiple database technologies

## Enterprise Search
- GDPR, HIPAA compliant. Recovers ~10 hrs/week/employee
- ADaaS (Airdrop-as-a-Service) for custom connectors
- Consumption-based pricing tied to MAU

## Agent Studio (coming soon)
- No-code, low-code, high-code agent building
- Agents inherit Memory context automatically
- Audit trail, versioning with rollback, enterprise security
